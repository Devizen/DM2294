WASD - move forward, left, backward, right
Arrow keys - pitch, yaw camera
Backspace to activate controls editor
Z for crouch
LMB for Shoot
R for Reload
Scroll for change weapon
Space for jump
Q change weapon
Left shift ro run
G to reset; continue game if screen is game over.

Features:
Minimap that shows enemy when a certain distance is reached.
Random spawning of enemies; 4 A.I states (IDLE, ALERT, DEATH and RECOVERY)
2 Different type of Weapons.
Controls Manager by press backspace.
Audio for game.
Highscore system that store the highest score reached.
Power-ups to help the player survive.
